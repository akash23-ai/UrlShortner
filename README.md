## Start The Application Locally
- Go to server Do 
  ```javascript
    npm install
  ```


- Create a .env inside server Directory
    ```
    MONGODB_URL = "Your MongoDb url"
    PORT = 3001
    ```
    
- Start the Server
    ```
    npm start
    ```

- Go to Url Client and Do 
    ```javascript
     npm install
    ```

- Start The Application
  ```
   npm run dev
  ```
